Contract Document Comparator

合同扫描件 vs 原版文档 — 字段级差异检测，自动告诉你哪里被改了

Python 3.10+ MIT License Beta / Pilot OCR + LLM

GitHub 快速开始 企业试点

核心应用场景

📄 合同扫描件核对

盖章合同扫描件（PDF/图片）与原始 Word 文档逐字段比对，自动检测金额、日期、条款等关键要素差异。

📊 Excel 附件核对

支持单元格级差异分析、行插入/删除检测、多 Sheet 比对，精准定位数据变更。

🔍 审计 / 招投标复核

全文差异追踪 + LLM 语义分析，自动评估差异风险等级（高/中/低），辅助人工复核决策。

功能特性

能力 说明
多引擎 OCR RapidOCR（默认）→ EasyOCR → Tesseract，自动降级
智能字段提取 15+ 正则模式，中文大写金额解析，全角/半角归一化
多维比对引擎 字段级匹配 + 数字容差 + 关键词重叠度 + 结构对齐
LLM 语义分析 Ollama（本地）+ Claude API，区分实质性差异与格式差异
6 种导出格式 TXT · JSON · DOCX（红线标注）· XLSX · PDF（A4）· ZIP
三合一界面 CLI · Streamlit Web（:8501）· FastAPI REST（:8080）
行业预设 通用 · 建设工程 · 租赁 · 采购 · 劳动合同
安全体系 API Key HMAC 认证 · RBAC 角色控制 · 审计日志 · 输入消毒

快速开始

# 克隆仓库
git clone https://github.com/contract-comparator/contract-comparator.git
cd contract-comparator

# 安装依赖
pip install -r requirements.lock

# 比对合同扫描件与原版文档
python main.py --word 原版合同.docx --pdf 扫描件.pdf

# 启动 Web UI
streamlit run app_streamlit.py
